## ğŸ”¹How to Use

1. Go to the **Extensions** menu in SillyTavern.
2. Find the **Welcome & Idle Voice** settings panel.
3. **Startup Sound:**
- Check "Enable Startup Sound".
- Enter the path to your audio file (e.g., `audio/welcome.wav`).
- Click "Test" to verify.
4. **Idle Sound:**
- Check "Enable Idle Sound".
- Enter paths for up to 3 different audio files (e.g., `audio/idle1.wav`).
- The extension will randomly pick one when the timer triggers.
5. **Configuration:**
- Set the **Idle Timeout** (minimum 60 seconds) and click "Apply".
- Adjust the **Volume** slider to your preference.
